Introduction

Noise level monitoring is required at places where low noise is a desirability. This has been implemented easily by using embedded systems in our project. This website is an integral component of our project that aims to create a mesh network of Arduino based sound sensors for the purpose of sound detection. These sensors ping their location in the event of a high intensity sound activity. This information is relayed along the Arduino nodes to the central Raspberry Pi node which uploads this data directly to this website hosted on the Baadal server.

sd

Objectives

  1. To create a wireless sensor network in the campus for noise level monitoring

  2. Model the network as a dedicated mesh of sound sensors deployed on Arduino nodes with a central Raspberry Pi node

  3. Record and maintain the data computed on the Raspberry Pi to the server, in the event of a noise activity

  4. Use this website to display the campus map with a blip at the location of the ping source.

sd

Method

  1. The sound module(KY-037) detects the presence of noise.

  2. Arduino UNO board (edge node) pings the sensor location through XBee modules.

  3. The signal is carried in a relay along edge nodes of mesh network.

  4. The signal finally reaches the Raspberry Pi (RPi) module (sensor node).

  5. The RPi module uploads the data on a server

  6. Blip at the location of the high intensity sound activity on the institute map.
